## Prerequisites
```

@emotion/react

@emotion/styled

@material-ui/core

@mui/material

@testing-library/jest-dom

@testing-library/react

@testing-library/user-event

dotenv-webpack

react

react-dom

react-router-dom

react-scripts

react-select

validator

web-vitals
```

## Installation

`npm install`

## Running the Frontend

`npm start`
